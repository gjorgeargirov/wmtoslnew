Help & Documentation

Getting Started

  1. Upload your WebMethods ZIP package
  2. Configure migration settings (environment, naming conventions)
  3. Click "Start Migration" to begin the process
  4. Download the results when complete

Frequently Asked Questions

What file formats are supported?

Only .zip files up to 100MB are supported. The ZIP should contain your WebMethods package structure.

How long does migration take?

Migration time varies based on package size, typically 2-10 minutes for standard packages.

Can I migrate multiple packages?

Yes, you can run multiple migrations. Each migration is tracked in your dashboard.

What happens to my original files?

Your original files are not modified. The migration creates new SnapLogic pipelines.

Troubleshooting

  • Migration fails: Check file format and size limits
  • Slow upload: Check your internet connection
  • Missing results: Wait for completion notification or check dashboard
